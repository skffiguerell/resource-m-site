# Resource M Site

## 项目简介

**resource-m-site** 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态 HTML 资源的集合与备份空间。  
每个页面均为独立文件，不依赖后端服务，可直接在浏览器中打开查看。

## 目录结构

```
resource-m-site/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── ...
```

- `pages/`：按需组织子目录，每个子目录对应一个独立页面或页面组。
- `assets/`：公共资源文件，可根据页面需要自行引用。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，无后端交互。
- 页面内容、样式、脚本均包含在对应文件或引用的资源中。
- 归档时不保证所有页面均处于最新或完整状态，部分页面可能为历史版本或临时备份。
- 页面之间无关联，可独立查看与使用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 下的任意 HTML 文件即可查看。
3. 如需部署到静态托管服务，可将整个仓库内容上传至服务器。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改的页面会直接提交至对应目录。
- 不提供持续集成或自动化部署保障。
- 页面内容可能因时间推移与原始来源存在差异，请以仓库内文件为准。
- 欢迎提交 Issue 或 Pull Request 进行内容修正或补充，但需确保不引入特定站点信息或商业推广。
